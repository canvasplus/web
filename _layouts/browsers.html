Other Browsers

Canvas+ is made to natively support Google Chrome and Firefox. However, many other browsers are compatible with Canvas+, though not officially supported.


Choose your Browser:



Google Chrome

We officially support Google Chrome. You can download Canvas+ for Google Chrome from the Chrome Web Store.


Firefox

We officially support Firefox. You can download Canvas+ for Firefox from Mozilla Addons.


Microsoft Edge

We do not officially support Microsoft Edge, but Canvas+ is compatible with it. Bugs specific to this browser will not be adressed for the time being. You can download Canvas+ for Microsoft Edge from the Chrome Web Store.


Opera

We do not officially support Opera, but Canvas+ is compatible with it. Bugs specific to this browser will not be adressed for the time being. You can download Canvas+ for Opera from the Chrome Web Store, but you'll need this extension first.


Opera GX

We do not officially support Opera GX, but Canvas+ is compatible with it. Bugs specific to this browser will not be adressed for the time being, and be warned, this browser has many. You can download Canvas+ for Opera GX from the Chrome Web Store, but you'll need this extension first.


Vivaldi

We do not officially support Vivaldi, but Canvas+ is compatible with it. Bugs specific to this browser will not be adressed for the time being. You can download Canvas+ for Vivaldi from the Chrome Web Store.


Brave Browser

We do not officially support Brave, but Canvas+ is compatible with it. Bugs specific to this browser will not be adressed for the time being. You can download Canvas+ for Brave from the Chrome Web Store.


Safari

Canvas+ is not compatible with Safari. Unfortunately, this browser only uses the App Store to load extensions. We're planning to address this soon.
